

## About This Website

Website Ini Adalah website chatting online sederhana yang Dibuat Menggunakan Laravel 10 dan pusher sebagar broadcasting

## Installation guide

```
git clone https://github.com/dewacs25/ChattingOnline
cd ChattingOnline
```

```
composer install
```
```
cp .env.example .env
```
```
npm install & npm run dev
```
Start server :
```
php artisan serve
```
